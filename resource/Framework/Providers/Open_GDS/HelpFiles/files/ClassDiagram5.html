Class Diagram

The Class Diagram displays a static view of all classes and the relationships between them.


Select desired mouse tool by clicking on the buttons on the top of the window. Use the context help to see the meaning of each mouse tool selection.


Right click or double click on a class to change:

- class name

- add/modify attributes

- add/modify methods

- browse to the GOOP code of the class (if it exists)


Linking a class to the code makes it possible to synchronize to and from UML on each class. Right click on a class to select synchronization operations.


When at least one class is defined the code generation selections are enabled in the Tools menu.


This diagram can also be created automatically from existing GOOP code. Use the reverse engineer functionality from the Tools menu to do this.
