Sequence Diagram

The Sequence Diagram displays a dynamic view of the system.


Select desired mouse tool by clicking on the buttons on the top of the window. Use the context help to see the meaning of each mouse tool selection.


The sequence diagram is used to show scenarios of interaction between objects.


Objects are shown as vertical bars. Method calls between objects are shown as arrows. Time flows from top to bottom of the diagram. Action bars can be used to show concurrency in the scenarios. See the context help for the Action Bar tool selection for more information.


First you need to add objects to be used and give them a name. It is normally a good idea to include the class name in the object name (for instance like this: "objName : Class Name") to make it easy to see which class each object is created from.


Each object is represented as a vertical bar.


When objects have been added you add an action bar to the object you want to add a method call from. Then select the Call mouse tool to add the method call.
