Finding the features

All tool features are integrated in the LabVIEW project environment. These are the different places you will find the tool features:

  1. right click on MyComputer>>New,
  2. right click on an created class; ClassName.lvclass>>New,
  3. right click on an created class; ClassName.lvclass>>GOOP,
  4. LabVIEW Tools menu; Tools>>GOOP,
  5. the green tool button.


Below is a list of where all the features can be found in the project explorer.


Create New Class

My Computer>>New>> GOOP Class and ClassName.lvclass>>New>>Create Derived Class

Create New Method

ClassName.lvclass>>GOOP>> Add Method

Add Design Pattern

ClassName.lvclass>>GOOP>> Add Design Pattern

Edit Class Icon

ClassName.lvclass>>GOOP>> Edit Class Icon

Edit Class Appearance

ClassName.lvclass>>GOOP>> Edit Class Appearance

Set Class Item Icon

ClassName.lvclass>>GOOP>> Set Class Item Icon

Rename Class

ClassName.lvclass>>GOOP>> Rename Class

Modify Class Inheritance

ClassName.lvclass>>GOOP>> Modify This Class Inheritance...

Clone Class

ClassName.lvclass>>GOOP>> Clone Class

Debug class

ClassName.lvclass>>GOOP>> Debug Class...

Get Class Information

ClassName.lvclass>>GOOP>> Get class information...
